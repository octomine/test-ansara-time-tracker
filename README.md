## Тестовое задание для ANSARA - простой трекер времени для задач.

* Вводим задачу и нажимаем на иконку Play, начинается отсчёт времени по этой задачи. Показывается время начала и время завершения.
* Выполняемую задачу можно остановить, она спускается вниз.
* Выполненную задачу можно добавить вручную через иконку плюсика

## Быстрый старт

### Скрипты проекта

```bash
# Устанавливка зависимостей
yarn install

# Проверка TypeScript
yarn type-check

# Проверяка линтинга
yarn lint

# Запуск тесты
yarn test

# Запуск dev сервера
yarn dev
```

dev сервер открывается в браузере [http://localhost:3000](http://localhost:3000)
